# Task 2 – Phishing Email Analysis

## 🎯 Objective
Analyze a phishing email to identify spoofing, suspicious links, and social engineering traits used by attackers.

## 🛠️ Tools Used
- Visual inspection
- Screenshot highlighting
- (Optional) MXToolbox Email Header Analyzer

## 📩 Sample Overview
- Subject: Response required
- Claimed Sender: PayPal
- Actual Email: service-paypal@outlook.com
- Technique: Fear-based social engineering

## ⚠️ Key Findings
- Spoofed sender address using Outlook domain
- Urgent language prompting fast action
- Fake “Resolution Center” link to capture credentials
- No personalization — generic greeting
- High resemblance to legit PayPal template to appear authentic

## 📁 Included Files
- `phishing_sample.png`: Original phishing email screenshot
- `phishing_analysis.txt`: Summary of findings
- `screenshots/`: Folder with visual highlights (optional)

## 🧠 Learnings
- Understood common phishing tricks
- Practiced identifying red flags in suspicious emails
